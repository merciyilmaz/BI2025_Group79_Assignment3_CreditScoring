# BI2025_Group79_Assignment3_CreditScoring

**Repository URI:** [https://github.com/merciyilmaz/BI2025_Group79_Assignment3_CreditScoring](https://github.com/merciyilmaz/BI2025_Group79_Assignment3_CreditScoring)

This repository contains the data preparation pipeline and visual artifacts for the Business Intelligence (2025) course assignment. It serves as the primary provenance record for the data cleaning and feature engineering process.

## Project Structure
- `Notebook_part_1_grp_079.ipynb`: Full Jupyter Notebook containing data cleaning (outlier removal), binning, and Z-score scaling.
- `figures/` (or root):
    - `raw_outliers.png`, `cleaning_result.png`: Visualizations for outlier detection and cleaning summary.
    - `income_binning.png`, `binning_relationship.png`: Analysis of binned features and their correlations.
    - `scaling_result.png`, `cleaned_stats_table.png`: Final standardized features and descriptive statistics.
